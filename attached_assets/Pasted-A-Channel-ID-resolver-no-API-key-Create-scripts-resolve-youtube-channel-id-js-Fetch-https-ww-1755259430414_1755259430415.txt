B) RSS fetcher (fast path; latest ~15 videos)

Create scripts/fetch-youtube-rss.js:

Run the resolver to get channelId.

GET https://www.youtube.com/feeds/videos.xml?channel_id=CHANNEL_ID.

Parse <entry> items → { title, link (from <link href>), date (from <published>) }.

Write to assets/data/youtube.json (newest first).

Note: YouTube’s RSS is limited to the most recent items (~15); this is a platform limit.

D) Wire the YouTube page

In youtube.html:

Keep the page copy exactly as you wrote.

Add:

<ul id="youtube-list" class="essay-list"></ul>
<script>
(async () => {
  try {
    const res = await fetch('/assets/data/youtube.json', { cache: 'no-store' });
    const items = await res.json();
    const ul = document.getElementById('youtube-list');
    items.forEach(item => {
      const d = new Date(item.date || item.publishedAt);
      const mm = d.getMonth() + 1, dd = d.getDate(), yy = String(d.getFullYear()).slice(-2);
      const li = document.createElement('li');
      const a = document.createElement('a');
      a.href = item.link || `https://www.youtube.com/watch?v=${item.videoId}`;
      a.target = '_blank'; a.rel = 'noopener';
      a.textContent = `${item.title}  –  ${mm}/${dd}/${yy}`;
      li.appendChild(a); ul.appendChild(li);
    });
  } catch {}
})();
</script>


Reuse the .essay-list CSS from §0 so it matches your Essays list styling.
